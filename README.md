**Fake News Headline Generator Using Python (GUI Based)**

**1. Objective of the Project**
The objective of this project is to generate random fake news headlines using Python.
It demonstrates:

    Random data generation
    GUI development using Tkinter
    Looping structures (while loop)
    Dictionary data structures
    File handling
    User interaction
This project automatically creates humorous fake news headlines based on selected categories.
